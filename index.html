mac安装mysql

发表于 2017-09-01

我在vue2+koa2练习demo博客中第一次使用mysql，这里记一下安装过程。
官网https://www.mysql.com/downloads/下载mysql，有问题百度。下载后安装，安装过程中会有一个临时密码的提示，要注意保存一下，后面会用到。安装完毕进入系统偏好设置在最下面找到mysql图标并打开，点击Start MySQL Server启动mysql。命令行cd到/usr/local/mysql/bin，分别执行alias mysql=/usr/local/mysql/bin/mysql和alias mysqladmin=/usr/local/mysql/bin/mysqladmin,这两条命令是为了方便直接打开 iTerm 就可以运行mysql命令，而不是必须进入mysql安装目录才能运行。接下来，重置密码。执行命令mysqladmin -u root -p password ****是你的新密码。回车后提示输入密码，此时数次刚刚保存的临时密码，我的是)6m1d/m
Warning: Since password will be sent to server in plain text, use ssl connection to ensure password safety.说明已经重置密码成功。我在这里浪费了很多时间，以为这是错误提示，其实已经ok了。<!--more--> 登录mysql。命令mysql -u root -p`,然后输入刚刚重置的密码。然后登录成功会跳出提示信息

大意是说sql命令以;结尾，当前mysql版本是5.7.19。另外我们也可以通过sql命令select version();来查看版本。用exit;退出登录。

Vue组件中data为什么是函数？

发表于 2017-08-23

vue组件中的data对象的值必须是函数，why？为什么不像文档起步介绍里那样直接是一个对象、而是一个返回值为对象的函数？文档解释见https://cn.vuejs.org/v2/guide/components.html#data-必须是函数。我说一下我的理解。
组件中data对象的值写的时候是一个返回值为对象的函数，但实际上在引用组件的时候，该函数被调用并且将返回值再次赋值给data，即将data的值由函数赋值为对象。data是组件对象实例的一个属性，data的属性值为对象，并且data的属性值值可以被代理到组件对象实例上，说明data对象内的属性是组件对象构造函数的prototype上的属性（未完待续）
