What Is VNC

vnc(Virtual Network Computing)是虚拟网络计算机的缩写。VNC 是一款优秀的远程控制工具软件，由著名的 AT&T 的欧洲研究实验室开发的。VNC 是在基于 UNIX 和 Linux操作系统的免费的开源软件，远程控制能力强大，高效实用，其性能可以和 Windows 或 MAC 中的任何远程控制软件媲美(摘自http://www.cnblogs.com/Yuanbangchen/p/5795728.html)。

安装远程连接软件

连接远程服务器，用命令apt-get install vnc4server安装vncserver

安装图形界面

命令apt-get install xfce4

启动vncserver

命令vncserver :1然后会提示输入一个密码，这个密码是8位的，如果超过8位则截取前8位。然后我们用命令vncserver -kill :1来把该进程kill掉，进行下一步。

在本地的电脑安装vncviwer

在本地电脑下载安装vncviwer https://www.realvnc.com/en/connect/download/viewer/macos/，安装好在地址栏输入远程服务器公网ip出现了连不上的问题。

因为vnc服务需要开放tcp协议下的端口5900-5903端口支持。百度到这样的解决方案：vi /etc/sysconfig/iptable,

我突然想到阿里云后台安全组设置那里可以设置端口的开放情况，现在只开放了上面这三个端口，怪不得vnc连不上。于是到阿里云后台新增了安全组规则，开放了tcp协议端口5900-5906，设置完成后到本地vnc viewer客户端地址栏输入服务器ip进行连接
